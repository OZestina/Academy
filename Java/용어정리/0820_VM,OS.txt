1. virtual machine이란(가상머신)
- 가상으로 컴퓨터를 만들 수 있는 프로그램

2. vm에서 image 파일이란?
- 가상환경에서 사용할 프로그램이 설치되어 있는 원본 파일

3. instance란?
- 가상환경에서 실제 사용할 프로그램 파일

4. 집으로 image 파일을 가지고 가서 사용하려고 합니다. 어떻게 하면 될까요?
- image를 생성해서 (꾸워서) 해당 instance를 집 컴퓨터에 설치한다(가상 시스템 가져오기)

5. vbox와 유사한 프로그램의 이름은?
- VMware

6. 가상환경 프로그램을 왜 사용할까요
- 하나의 컴퓨터에서 다른 운영체제나 컴퓨터를 구동해볼 수 있게 해주니까요
- 호스트 컴퓨터에서 수행하기 위험한 작업이나 수행할 수 없는 운영체제를 실행해보기 위해

7. 가상 시스템 내보내기/가져오기란?
- 내보내기: 시스템을 image로 만들어서 (배포할 수 있게) 저장하는 것
- 가져오기: 만들어진 image를 설치하는 것

8. 운영체제는 뭐가 있을까요?
- 윈도우, 리눅스, 맥, 유닉스, 도스
