import java.lang.*;

Integer, System은 java.lang에 있는 class이지만,
Java 실행 시 import java.lang.*; 이 자동으로 실행돼 별도의 import 없이 클래스 이름만으로 사용할 수 있다.

.* 은 java.lang 하위의 여러 개를 모두 아우를 수 있는 것.
