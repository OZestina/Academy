12.TCP와 UDP의 차이

TCP와 UDP의 차이
tcp : 인터넷상에서 데이터를 메시지 형태로 IP와 함께 사용하는 프로토콜. 인터넷 환경의 기본으로 사용.
udp : 인터넷상에서 전송 신호 없이 일방적으로 데이터를 전달하는 통신 프로토콜. 데이터를 데이터그램 단위로 처리한다.
신뢰성 : tcp, 연속성 : udp

TCP/IP 4단계 구분, 설명, 예
네트워크 엑세스, 인터넷 계층, 전송 계층, 응용 계층
네트워크 엑세스 : 네트워크, 디바이스 드라이버 등 하드웨어적인 요소와 관련된 계층. MAC
인터넷 계층 : IP패킷 헤더를 붙여 전송. IP
전송 계층 : 네트워크 양 단의 송수신 호스트 사이에서 전송. TCP, UDP
응용 계층 : 응용 프로그램이 네트워크서비스, 웹서비스 등을 할 수 있도록 표준적인 인터페이스를 제공. FTP, DNS, SSH
