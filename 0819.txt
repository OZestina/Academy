리눅스
pwd: 현재 위치 프린트

ls : 현재 폴더 내 디렉토리/파일 보여줘

ls -a: (all) 숨김 폴더/파일까지 다 보여줘라

la -al: 각각의 폴더/파일 세부사항도 나온다
rwx : read, write, execute - 권한을 나타낸다
rwxrwxr-x: 해당계정 + 내그룹+다른그룹
zestina zestina : 처음 2개는 zestina의 권한, 마지막 하나는 게스트의 권한 (read, execute만 되고 write 안됨!)

R W X
4  2  1
1: execute (실행)
2: write (쓰기)
3: w+x
4: read (읽기)
5: r+x
6: r+w
7: r+w+x

권한 변경
>chmod (권한3종) (디렉토리명)
(change mod) 700 
chmod 777 f* :  f로 시작하는 디렉토리의 권한을 모두 업데이트 (*r : r로 끝나는거

프로그램 -version : 설치된 프로그램의 버전 정보 출력
(java -version)

cd: change directory
> cd / : root로 이동
> cd .. : 상위 디렉토리로 이동
> cd ../.. : 상위 상위 디렉토리로 이동
> cd (디렉토리명) : 디렉토리로 이동
> cd ~ : 내 디렉토리로 이동 (/home/ozestina)
  ~(tilde, 틸드)

폴더만들기
> mkdir 폴더명 폴더명 폴더명 (폴더 여러개 생성 가능)
(make directory)

rm -rf (디렉토리/파일명) : 삭제
rm dir (디렉토리) : 디렉토리 삭제

mv (디렉토리) 기존폴더명 새폴더명 : 이름 변경
mv (디렉토리) 기존폴더명1 기존폴더명2 : 위치 이동
cp -r 기존폴더명 새폴더명 : 폴더 복사

tree (디렉토리명) : 디렉토리 내의 파일/디렉토리를 트리모양으로 보여줘라

sudo : (window) super user 명령어 (관리자 권한으로 실행인듯?)

touch 파일명 : 파일 생성
cat 파일명.확장자 : 파일의 내용을 출력
cat ./win.txt : 현재 위치에 있는 win.txt를 출력해라 (가끔 폴더 내에 있는 파일을 못읽을때 이렇게 지정)
cat > 파일명.확장자 : 파일에 내용 쓰기 (중지는 ctrl+c)

clear : 화면 삭제 (윈도우는 cls)
  > 단축키: ctrl + l

ping 8.8.8.8

ctrl + c : 터미널에서 돌아가는 프로세스 종료
